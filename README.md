# Reactulator
A fun little calculator to learn about React.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Requirements
Should be using node version >=8.10

Run `node -v` from terminal to display your version or node 

## Install   
`yarn install`

## Build
`yarn build`

## Run
`yarn start`

Navigate to http://localhost:3000 in your browser. 

## Test
`yarn test`
